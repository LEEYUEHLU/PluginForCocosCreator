1. <使用 npm 安裝 Inversify>

2. <新增 inversify.config.ts>

/////在inversify.config.ts檔案裡////////////

3.<引入inversify模組 :> 
	import { Container, ContainerModule, interfaces } from "inversify";

4.<產生一個container : > 
	const myContainer = new Container();

5.<用inversify的模組來定義物件相依條件 : >

	(1.)	let DTBind = new ContainerModule(
   	(2.)	 	(bind: interfaces.Bind, unbind: interfaces.Unbind) => {
        (3.)		bind(TestDI).toSelf();});
  
  其中第三行 "bind(TestDI).toSelf();" 向IoC Container 定義了 "TestDI" 類別的相依模式。

6. <使用load()來載入定義好的模組:>
	myContainer.load(DTBind )
	
7.以上完成 DI 的註冊
