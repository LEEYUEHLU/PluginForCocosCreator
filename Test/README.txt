Set up enviroment:

	1. Put this folder("Test") under project foler (ProjectFolder / Test)
	2. Put "package.json" under project folder.(ProjectFolder / package.json)
	3. Open cmd and navigate to project folder.
	4. type "npm install" in command line.
	5. type "npm test" in command line, than you will see  "mochawesome-report" was generated.(ProjectFolder/mochawesome-report).
	6. check out the report file by navigating to "ProjectFolder/mochawesome-report/mochawesome.html".

Writing Tests:

	1. add file under "Test" folder, naming convention is XXX.spec.ts.
	2. writing tests as example file HelloTest.spec.ts.
